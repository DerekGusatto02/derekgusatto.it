Stickman Run is a project developed in Java during the fourth year of high school for the computer science diploma. The project was carried out with W. Zanella between December 2019 and February 2020.

The aim of the project was to create a game using Java and for graphics the Standard Widget Toolkit, the chosen IDE was Eclipse
