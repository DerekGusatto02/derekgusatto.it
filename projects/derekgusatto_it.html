The portfolio website is composed of several sections that present personal information, projects, and contacts.

Overview of the main sections and their contents:

  • Homepage: contains a brief introduction and a background image
  • About me: contains personal information, skills, and education
  • Projects: contains a list of completed projects with a brief description and a preview image
  • Contacts: contains a contact form and information to get in touch with me

Technologies used

  • HTML5: Used for the structure of the web pages.
  • CSS3: Used for the style and layout of the pages.
  • JavaScript: Used for interactive functionalities such as form validation and menu management.
  • PHP: Used to dynamically include HTML content and handle message sending from the contact form.

Accessibility

The website has been designed with particular attention to accessibility, including the following features:

  • Responsive Design: Use of media queries to adapt the layout to different screen sizes.
  • ARIA Navigation: Use of ARIA attributes to improve navigation for users with disabilities.
  • Color Contrast: Use of high contrast colors to facilitate reading.
  • Readable Text: Use of readable text sizes and adequate spacing to facilitate reading.
  • Form Validation: Validation both client-side (JavaScript) and server-side (PHP) to ensure the integrity of the entered data.

Development Strategies

The main development strategies adopted include:

  • Content Separation: Use of HTML for structure, CSS for presentation, and JavaScript for behavior.
  • Modularity: Division of code into separate files to improve maintainability.
  • Code Cleanliness: Use of comments and consistent formatting to make the code readable and understandable.
